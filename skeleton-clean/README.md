# Call Analyzer

Бот для аналізу клієнтських дзвінків автосервісу: транскрибація аудіо
з Google Drive і автоматична оцінка роботи менеджера через LLM.

## Статус: 🚧 У розробці

## Технології

- Python 3.10+
- Google Drive / Sheets API
- faster-whisper (транскрибація локально)
- Ollama + Qwen 2.5 (LLM-аналіз локально)

## Запуск

Інструкції додам після завершення MVP.
